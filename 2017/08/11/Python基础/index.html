Python基础

发表于 2017-08-11   |   分类于 Python   |  

基础语法

常量/变量

常量：一旦被赋值，就不可再改变。Python不存在常量

字面常量：一个单独出现的量，未赋值给任何变量或常早

变量：是一个名字，在赋值符号的左边，这个名字可以指代赋值符号右边的内容

类型系统

  • Python是强类型语言
  • Python是动态语言

强类型：指不同类型之间不能相互计算
动态类型：变量可以重新赋值为其他类型

基本类型

  • int
  • float
  • bool -> True or False
  • None -> None

算术运算

算术运算符只能对int和float运算

  • +
  • -
  • *
  • / -> Python3自然除 and Python2取整数
  • // 取整除
  • ** 求幂
  • % 求模

赋值运算符可以和算术运算符一起使用。

比较运算符

  • >
  • <
  • ==
  • !=

  • =

  • <=

逻辑运算符

逻辑运算符的操作数都是bool类型或者可以隐式转化成bool类型的类型，反回值都是bool类型

  • and
  • or
  • not

位运算符

  • <<

  • >

  • &
  • |
  • ^

顺序结构

  • 顺序 if
  • 分支 if .. else
  • 循环 while、for in

提前终止

  • break 终止
  • contine 跳过

内置数据结构

列表

列表是一个序列，用于顺序的存储数据。

定义与初始化

元组常用操作

字符串及其操作

字符串是可迭代对象
字符串是不可变的

字符串常用操作

字符串格式化

字符串与bytes

bytes是文本序列
bytes是字节序列

  • 文本是有编码（utf-8,gbk,GB18030等）
  • 字节没有编码这种说法
  • 文本的编码指的是字符如何使用字节来表示

python3字符串默认使用utf-8编码

bytearray

bytearray是bytes的可变版本
str和bytes是不可变的

线性结构与切片

特征:

  • 可迭代
  • len获取长度
  • 可以使用下标操作符通过索引访问
  • 可以切片

解构与封装

Python3解构的变化

  • 元素按照顺序赋值给变量
  • 变量和元素必须匹配
  • 加星号变量，可以接受任意个数的元素
  • 加星号的变量不能单独出现

结构是支持多层次

非常复杂的数据结构，多层嵌套的线性结构的时候，可以用解构快速提取其中的值

集合与集合操作

数学意义上的集合

集合不能修改单个元素
集合不是性线结构，集合元素没有顺序

成员运算符

in
not in

用天判断一个元素是否在容器中

集合运算

是否超集（isuperset）
用来判断前面是否是后面的超集。

是否子集(issubset)
用来判断前面是否是后面的子集。

是否不相交（isdisjoint）
用来判断两个 是否是不相交

字典及其操作

字典遍历

values()反回一个可迭代对象，元素是字典的所有value
items()反回一个可迭代对象，元素是字典的所有key
*keys()反回一个可迭代对象，元素是字典所有的key

字典的限制

字典的key不能重复
字典的key需要可hash

默认字典

default初始化的时候，需要传入一个函数，这个函数叫工厂函数，当我们使用下标访问一个key的时候，如果这个key不存在，defaultdict会自动调用初始化时传入的函数，生成一个对象作为这个key的value

有序字典

解析式

  • 代码简洁，可读性强
  • 效率比普通迭代稍高

迭代器

有iter对象叫可迭代对象

iter函数可以把一个可迭代对象转化为迭代器

去重的三种方法

函数

函数是python里组织代码最小单元

可变参数的两种形式:

  • 位置可变参数：参数前加一个星号，构成元组，传参只能以位置参数的形式
  • 关键字可变参数：参数名前加两个星号，构成字典，传参只能以关键字参数的形式

通常来说：

  • 默认参数靠后
  • 可变参数靠后
  • 默认参数和可变参数不同时出现

参数解构的两种形式：

  • 一个星号解构的对象：可迭代对象 解构的结果：位置参数
  • 两个星号解构的对象：字典 解构的结果：关键字参数

keyword-only 参数

keyword-only参数可以和默认参数一起出现，不管它有没有默认值

函数反回值

一个函数有多个return语句，当return反回结果并结束函数

作用域

局部作用域
全局作用域：global

匿名函数

  • lambda来定义
  • 参数列表不需要用小括号
  • 冒号不是用来开启新语句块
  • 没有return，最后一个表达式的值即反回值

匿名函数通常用于高阶函数的参数，当此函数非常短小的时候，就适合使用匿名函数

生成器

带yield语句的函数称之为生成器函数，生成器函数的反回值是生成器

  • 生成器函数执行的时候,不会执行函数体
  • 当next生成器当前代码执行到第一个yield会弹出值，并且暂停函数
  • 再次执行next生成器时，从上次暂停处开始往下执行
  • 当没有多余yield时候，抛出StopIteration异常，异常value是函数的反回值

面向对象

面向对象是一种编程范式

OOP的世界观：

  • 世界是由对象组成
  • 对象具有运动规律和内部状态
  • 对象之间可以相互作用

OOP的特征：

  • 封装
  • 继承
  • 多态

类方法

访问控制

所有双下划线开始，非双下划线结尾的成员都是私有成员
当单个下划线，是一种惯用法，标记此成员为私有，但是解释器不做任何处理

类的继承

  • 凡是公有的都能继承
  • 凡是私有的都不能继承

方法重写

可以通过super()方法调用父类的方法

C3算法

如果多继承通过MRO方法查询顺序

  • 本地优先：自己定义或重写的方法优先，按照继承列表，从左到右查找
  • 单调性：所有子类，也要满足查找顺序

python通过C3算法来确定是否满足mro的两个原则
