Discourse + godaddy 配置

发表于 2018-01-06   |   分类于 Linux运维   |  

上次搭建的网盘大伙都没有用,最初以为大伙不要博客或者论坛形式,所以就找了网盘,这个月低开会又提到了交流问题,所以找了一圈看上了Discourse,坑非常的多自己踩了几天终于完美的解决了。
Discourse用Docker来安装主要重点是SMTP配置,然后找了一圈有人用mailgun发送邮箱,Discourse不管是安装还是注册会员一定需要邮件就对了,包括后续的管理。
刚申请了mailgun测试可以发信不过比较麻烦的是要绑定信用卡,可我没有这种东西,所以麻烦将就麻烦,需要添加信任的邮件才能收到。

https://github.com/discourse/discourse/blob/master/docs/INSTALL-cloud.md

系统环境:
Ubuntu 16

安装Docker
wget -qO- https://get.docker.com/ | sh

安装git
apt-get install git

不管是在外网还是在内网域名必须要

https://godaddy.com/
用国外的不需要备案

注册短信接口
https://app.mailgun.com/

提供测试接口,一个二级免费的域名只能指定信任邮箱接收,其它发送不出去。

这里想升级为1千条免费短信需要绑定信用卡,这里使用taobao的虚拟信用卡 30块搞定.

安装：
root权限操作

mkdir /var/discourse

cd /var/discourse

激活管理员邮箱就行.
