函数

发表于 2016-12-27   |   分类于 Python   |  

函数定义

通过def定义一个函数，fn()是一个函数名，def的缩写是define

函数名后面的x,y是参数，return表示函数的反回值,当定义函数的时候并不会执行函数体，当调用的时候，才会执行其中的语句块,传入的参数必须要和定义的参数相匹配，否则抛出TypeError异常

参数

参数可以有默认值，当一个参数有默认值时，调用时如何不传递此参数，会使用默认值，默认值要在不带默认值后面

可变参数

参数加*号，表示这个参数是可变的，也就是可以接受任意多个参数

参数前加 **表示这个参数是可变的，可以接受任意多个数，这些参数构成一个字典，此时只能通过关键字参数传参

可变参数有两种形式：
位置可变参数：参数名前加*个星号，构成元组，传参只能以位置参数的形式
关键字可变参数：参数名前加**个星号，构成字典，传参只能以关键字的形式，关键字可变参数不允许在普通参数之前，否则会抛出异常

位置可变参数可以在普通参数之前，但是在位置可变参数之后的普通参数变成了keyword-only参数

三条规则：

  • 默认参数后
  • 可变参数靠后
  • 默认参数和可变参数不同时出现

参数解构

参数解构加一个星号，可以把一个可迭代对象解构成位置参数

参数解构的两种形式

*一个星号，解构的对象：可迭代对象 解构结果：位置参数

**两个参数，解构的对象：字典 解构的结果：关键字参数

位置参数解构 加一个星号， 可以把可迭代对象解构成位置参数

keyword-only 参数

函数反回值

return 语句除了返回值之外，还会结束函数， return之后的语句将不会被执行

一个函数可以有多个return语句， 执行到哪个return由哪个return返回结果并结束函数

当函数没有return语句时，反回None

函数嵌套

函数中可以调用函数

作用域

作用域是一个变量的可见范围叫做这个变量的作用域

当在函数中的一个变量称为局部作用域，通过关键字global来定义称为全局作用域，函数外也可以调用。

上级作用域对下级作用域可见

nonlocal 关键字

nonlocal 关键字用于标记一个变量由他的上级作用域定义， 通过nonlocal标记的变量， 可读可写

函数的执行流程

main # 全局作用域
f1() # 开启
f2()
f3()
f4()
f5()
main

当调用函数的时候， 解释器会把当前现场压栈，然后开始执行被调函数， 被调函数执行完成，解释器弹出当前栈顶，恢复现场

递归函数

自己调用自己
递归函数必须要有退出条件
为了保护解释器，python最大递归深度有限制

匿名函数

lambda 来定义匿名函数
参数列表不需要用小括号
冒号不是用来开启新的语句块
没有return，最后一个表达式的值即反回值

第一对括号用来表示优先级，第二个表示调用函数
匿名函数只能写在一行上，也有人称之为单行函数
匿名函数通常用于高阶函数的参数，当此函数非常短小的时候，就适合使用匿名函数。

生成器

带yield语句的函数称之为生成器函数，生成器函数的反回值是生成器

生成器函数执行的时候，不会执行函数体
当next生成器的时候，当前代码执行到之后的第一个yield，会弹出值，并且暂停函数
当再次next生成器的时候，从上次暂停处开始往下执行
当没有多佘的yield的时候，会抛出StopIteration异常，异常的value是函数的反回值

yield表示一个弹值
