PeepingTom

用于渗透的时候扫描C段的机器，然而你要去一个一个的打开查看是什么服务，这软件就是用来快速帮你截图查看用的。我记得Kali2下面也有类似的工具。

使用前需要安装：
PeepingTom
CURL
phantomjs：下载地址

参数：
-l 加载文本
-x 加载nessus、nmap扫描的xml文件
-s 加载一个目标，我觉得没有什么必要
-o 输出目录
-t 超时时间，默认则是8秒
-v 详细过程
-b 在浏览器中打开结果

ADS

把1.exe隐藏在1.txt文件中:
type c:\1.exe > 1.txt:1.exe

Windows7改变了这个功能。
给符号链接设置隐藏的文件属性：
attrib +h/L 1.exe

保持隐藏文件
mkdir \\?\c:\tmp\"..\"

把恶意文件放到该目录里：
move 1.exe "\\?\c:\tmp\.."

复制出来：
copy "\\?\c:\tmp\.. \1.exe"

好了，以上是对那本书想记下来的东西，其实这类工具书还是不推荐买的。其实自己真是作贱看到封面这么好看了目录会去买，看完后就各种喷。
